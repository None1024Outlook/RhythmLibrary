# RhythmLibrary

A Python tool library for obtaining rhythm game information, supporting data queries for multiple rhythm games.

## Supported games

### Supporting
- KALPA
- Phigros
- Rotaeno

### Plan support
- CHUNITHM
- MAIMAI DX
- Orzmic
- osu!
